As_mediabox

As_mediabox is a new media shadow box made by Appsaloon

Requirements: Jquery.

First add:

  • as_mediabox.js put in head after jquery
  • as_mediabox.css put in head
  • Add this script on the bottom of your page
    <script>
     as_mediabox.setScribdPubKey('scribd_publisher_id');
     as_mediabox.init();
    </script>
To start you need a link with a class mediabox.

Pdf viewer with Scribd

To view a pdf: add class scribd to the link and use the href for the link of the pdf file.

  • The first time you use the link your pdf will be uploaded to your scribd account and be processed. Later the scribd version of your pdf will be available.
  • Pdf's cannot be protected otherwise scribd cannot process them.
